Bulk Upload Queues
Upload Instructions

Upload a .xlsx or .csv file with the following columns:

  • Name: Display name for the queue
  • Code: Unique identifier for the queue
  • Type: Queue type (standard, delayed, dead-letter)
  • VNamespace: Virtual namespace for logical grouping
  • TTLQueue: Time to live in seconds (number >= 0, optional, default: 0)
  • AllowDuplicated: Allow duplicates (true/false, yes/no, 1/0, optional, default: true)
  • MaxAttempts: Maximum attempts (number > 0, optional, default: 1)
  • PriorityType: Priority type (normal/fair, optional, default: normal)
  • MaxPriority: Maximum priority level (1-100, optional, default: 1)
  • PriorityThresholds: For fair priority, comma-separated values per priority (e.g., "10,20,30", optional, default: 0 for each priority)
First 4 fields are required. Code must be unique within the tenant. File must include header row.
