  • DefaultQueueMessageDelayTime: Default message delay time in seconds (number >= 0, optional, default: 0)
  • QueueExpires: Queue expiration time in seconds (number >= 0, optional, default: 0)
  • AllowDuplicated: Allow duplicates (true/false, yes/no, 1/0, optional, default: true)
  • MaxAttempts: Maximum attempts (number > 0, optional, default: 1)
  • MaxQueueSize: Maximum queue size (number >= 0, optional, default: 0 = unlimited)
  • MaxDeliveringMessages: Maximum messages in delivering state (number >= 0, optional, default: 0 = unlimited)
  • PriorityType: Priority type (normal/fair, optional, default: normal)
  • MaxPriority: Maximum priority level (1-100, optional, default: 1)
  • PriorityThresholds: For fair priority, comma-separated values per priority (e.g., "10|20|30", optional, default: 0 for each priority)
